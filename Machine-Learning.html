Predicting Maximum Drawdown in Indian Equity Markets

A comparative analysis of machine learning models using the Nifty 50 Index (2008–2026). UCL MSc Dissertation.

Institution

UCL

Expected

August 2026

Status

In Progress

This project is currently in progress. Full findings, model outputs, and the complete dissertation will be published here upon submission in August 2026.

Overview

This dissertation investigates whether machine learning models can outperform traditional statistical methods in predicting maximum drawdown — the peak-to-trough decline — in the Indian equity market, using the Nifty 50 Index as the primary data series.

The research covers the period 2008 to 2026, deliberately including the 2008 global financial crisis, the 2020 COVID-19 crash, and the subsequent recovery, to stress-test model performance across distinct market regimes.

Model Framework

1

Linear Regression

Baseline model. Establishes a performance floor against which all subsequent models are benchmarked.

2

Ridge Regression

Regularised linear model. Tests whether penalising coefficient magnitude improves out-of-sample prediction.

3

Random Forest

Ensemble tree model. Captures non-linear relationships between engineered features and drawdown magnitude.

4

XGBoost

Gradient boosted trees. The most complex model in the ladder — evaluated for marginal gain over Random Forest.
